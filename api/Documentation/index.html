Text commands
Command Argument(s) Description Alias(es) Permissions required
By bot By you
poll Space-separated pairs (Text*), Title (Text+) Starts a poll (vote). Example of 1 pair: yes:green_check_mark: vote Add reactions, Embed links
insert Emoji (Text), Text to work on (Text+) Replaces spaces in the text with emojis 😀 just 😀 like 😀 this.
spoiler Text to work on (Text+) Generates an annoying piece of spoiler text.
rawspoiler Text to work on (Text+) Generates an annoying piece of spoiler text to copy and paste.
rawrawspoiler Text to work on (Text+) Similar to rawspoiler, but the recursion depth is 2.
reverse Text to work on (Text+) Reverses a piece of text.
emoji Text to work on (Text+) Turns text into regional indicators and other emoji symbols.
base From base, To base (Integer), Number (Text+) Turns text into regional indicators and other emoji symbols.
encode Algorithm (base16/32/64, blake2b/2s, sha1/224/256/384/512), Text (Text+) Encodes/hashes the piece of text.
decode Decodes the piece of text. Note: blake- and sha- hashes are not available!
length Text (Text+) Analyses the length and most common characters in the text.
spellcheck Text (Text) Spellchecks a word based on the QWERTY keyboard layout.
case Text (Text) Makes the piece of text all-lowercased, all-uppercased or in title casing.
