SuperBot Official Documentation

Basic commands
Command Argument(s) Description Alias(es) Permissions required
By bot By you
help Sends a list of commands. Does not support in-depth help. commands Embed links
inter_help Sends in-depth help with the help of select menus.
invite Sends a link for you to invite the bot to your own server.
prefix Reminds you of the prefix of the bot (Equal sign =).
ping Checks whether the bot is online and tells you the response speed. statusonline
botpurge No. of messages to delete (Integer) Deletes the most recent message(s) sent by the bot in the current channel. Manage messages

Moderation commands
Command Argument(s) Description Alias(es) Permissions required
By bot By you
kick Member (Member), Reason (Text+) Kicks a member. Kick members
ban Bans a member. Ban members
unban User (User), Reason (Text+) Unbans a member.
slowmode Delay (Text+) Sets slowmode for the current channel. Manage channels
Delay (Text+), Channels in the server (Channels*) Sets slowmode for one or more channels.
purge Number of messages to delete (Integer) Deletes the most recent message(s) in the current channel. Useful for cleanup. Manage messages
purgeuser Number of messages (Integer), User (User) Deletes the most recent message(s) in the current channel sent by the user.
purgeregex Number of messages (Integer), Regex (Text+) Selectively deletes the most recent message(s) in the current channel based on a regular expression, python script or both. See Appendix A2.
purgepy No. of messages (Integer), Python script (Text+)
purgepygex No. of messages (Integer), Regex (Text), Python script (Text+)
makeinvite Valid duration (Text), Max. users (Integer) Generates an invite for the server. Defaults to infinite users if not set. Create invite


Text commands
Command Argument(s) Description Alias(es) Permissions required
By bot By you
poll Space-separated pairs (Text*), Title (Text+) Starts a poll (vote). Example of 1 pair:yes:green_check_mark: vote Add reactions, Embed links
insert Emoji (Text), Text to work on (Text+) Replaces spaces in the text with emojis 😀 just 😀 like 😀 this.
spoiler Text to work on (Text+) Generates an annoying piece of spoiler text.
rawspoiler Generates an annoying piece of spoiler text to copy and paste.
rawrawspoiler Similar torawspoiler, but the recursion depth is 2.
reverse Reverses a piece of text.
emoji Turns text into regional indicators and other emoji symbols.


Information commands
Command Argument(s) Description Alias(es) Permissions required
By bot By you
color #HEX or R G B or Decimal (Text+) Shows information about a color. colour Embed links
simplecolor Color name (Text) Shows information about a gradient or color. simplecolour
translate Language code (Text), Sentence (Text+) Translates the text to the language.
definition Word (Text) Looks up the word in a dictionary.
rtimer Duration (Text) Starts a timer with beautiful regional indicators. See Appendix A3.
ttimer Starts a timer in plain text. See Appendix A3.
terminate Timer ID (Text) Terminates a timer given its code. See Appendix A3.
unscramble Alphabets (Text) Unscrambles a piece of text. Useful when playing Scrabble.
unicode Query (Text+) Looks up unicode characters based on their names.
random Lower (Integer), Upper (Integer) Draws a random integer between the two arguments.
choice Possible choices (Text*) Draws a random choice out of the arguments.
raffle Lower (Integer), Upper (Integer), Times (Integer) Draws random integers between the bounds for multiple times.
pick Similar to raffle, but entries do not repeat.
