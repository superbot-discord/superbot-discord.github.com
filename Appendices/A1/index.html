SuperBot Documentation - Appendix 1

Sending Embed messages withembed

=embedtakes in 11 arguments spread over 11 lines:

=embed Title
Description
Color
Title URL
Author
Footer
Author URL
Image Link
Thumbnail Image Link
Author Image Link
Footer Image Link

Adding fields

For every field, insert 3 lines to the bottom of your command:

Field Title
Field Description
Field Inline

Field Inline defines whether the field can be placed in a horizontal row with other fields. Useyfor 'yes' and any other value for 'no'. This only has an effect if there are more than 1 field.

Using simplifiedsimpembed

=simpembedonly takes in 4 arguments for convenience:

=simpembed Title
Description
Color
Image Link

Similar toembed, you can insert field by appending lines to the command.

Skipping things withquickembed

=quickembedhas the same arguments assimpembed, but the order is different:

=quickembed Title
Color
Image Link
Description

At the price of not being able to add fields, you don't have to use{{{newline}}}for lines. You can simply append lines to the back.

Notes

Color only takes in a decimal value! Use=colorto fetch the decimal value.
Images are larger than Thumbnails while author/footer images are extremely small.

If you leave a line empty, it will not be processed. If you leave the color empty, it will be automatic based on light/dark theme.

To add Masked URLS into embed/field description, use[Mask](URL), such as[Wiki](https://wikpedia.com).
To add a new line to embed/field descriptions, use{{{newline}}}.
