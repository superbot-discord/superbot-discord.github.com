What's on?

Entries on top are most recent.

Typingâ€¦

Ever used a complex command? Not sure whether SuperBot is down? We implemented a typing indicator for the youtube command. We'll apply this on other commands in a short time.

Arguments

It might be confusing if you type a command but SuperBot sends nothing. That's why we added missing-arguments detection, making it simple to understand what's wrong with the command.

Typos

To bring convenience to mobile users, SuperBot now supports typo correction for command names.

Count polls

After some development, SuperBot can finally count your polls automatically when your add or remove a reaction. If moderators accidentally removed the placeholder reaction, it will add the reactions back.

Split files

We're testing out split files on our bot! We've tested it on several commands are we're rolling this out. This would mean a significant drop in file size and deploy speed.

New IDLE

I had been editing files on GitHub website directly before, since the start of the bot. However, thanks to my Javascript teacher's recommendation, I started using Visual Studio Code. It'd be more efficient to handle bugs.
