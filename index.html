Typos

Beta • To bring convenience to mobile users, SuperBot now supports typo correction for command names. It tries to find a matching command even if it doesn't make sense, e.g.abcwould be processed asascii.

Count polls

Testing • After a long development process, SuperBot can finally count your polls automatically when your add or remove a reaction. If moderators accidentally removed the placeholder reaction, it will add the reactions back.

Split files

Testing • We're testing out split files on our bot! We've tested it on several commands are we're rolling this out. This would mean a significant drop in file size and deploy speed.
