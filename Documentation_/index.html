snipe {Boolean}

If the boolean is supplied, enables or disables sniping in the current channel. Otherwise, shows the 5 most recently deleted messages. Disabling snipe will not clear the deleted message database! If you want to do so, use clearsnipe.

Arguments

{Boolean} Whether you want to enable snipe or not.

Alias

N/A

Permissions

The bot requires the Embed Links permission.
If you supply the Boolean argument, you will need the Manage Channel permission.

Moderation Commands (Members)

kick [Member] {Reason}

Kicks the user out of the server once.

Arguments

[Member] The user to kick. Can be an ID, mention, Name#Discriminator notation, name or nickname.

Alias

N/A

Permissions

The bot and you need the Kick Members permission.

ban [User] {Days} {Reason}

Bans the user and notify him via a DM. He will not be able to rejoin until someone unbans him.

Arguments

[User] The user to ban. Can be an ID, mention, Name#Discriminator notation or name.
{Days} Must be an integer between 0 and 7 inclusive. After the ban, Discord will delete X days worth of messages sent by the user. Defaults to 0 (don't delete any).
{Reason} The reason to ban the user. Shows up in the audit log.

Alias

N/A

Permissions

The bot and you need the Ban Members permission.

unban [User] {Reason}

Unans the user.

Arguments

[User] The user to unban. Can be an ID, mention, Name#Discriminator notation or name.
{Reason} The reason to ban the user. Shows up in the audit log.

Alias

N/A

Permissions

The bot and you need the Ban Members permission.

timeout [Member] {Duration} {Reason}

Timeouts the user. If the duration is not set or is zero, removes the timeout.

Arguments

[User] The user to unban. Can be an ID, mention, Name#Discriminator notation or name.
{Reason} The reason to ban the user. Shows up in the audit log.

Alias

N/A

Permissions

The bot and you need the Ban Members permission.
